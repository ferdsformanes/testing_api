
# REST Client - Quick Setup Guide

## Install
1. Open VS Code.
2. Go to Extensions (`Ctrl+Shift+X`).
3. Search **REST Client** and install the one by Huachao Mao. citeturn1search6

## Create Request File
1. Create a new file and save as `.http` or `.rest`.
2. Add a request, e.g.:
```http
GET https://api.github.com
```

## Send Request
- Click **Send Request** above the line. citeturn1search6

## Multiple Requests
- Separate requests using `###` in the same file. citeturn1search6

## Variables (Optional)
- Use `{{variableName}}` in requests and define them in settings. citeturn1search6
